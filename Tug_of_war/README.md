#1차원_문자_배열, #함수, #로테이션, #2인용게임

### __설명:__
2인용 줄다리기 게임. 1차원 문자 배열과 배열의 주소를 인자로 하는 함수, system 함수 연습 목적의 프로젝트. 
추가로, 교재에 나온 로테이션에 대한 이해를 돕기 위해 만든 프로젝트.

#### __<규칙>__
1. 줄 당기기: user1과 user2가 각각 ‘a’ 버튼과 ‘;’ 버튼을 연타.
2. 승리: 줄다리기 줄의 중심이 10의 위치에 있으면 user2의 승리이고 줄의 중심이 70의 위치에 있으면 user1의 승리.
   (줄의 길이: 81)
3. 항복: user1이 ‘q’를 누르거나 user2가 ‘p’를 누르면 항복 및 게임 종료.
4. 중단: 'x’는 게임 도중 종료.

#### __<조건>__
1. 줄다리기 줄은 1차원 배열로 표현한다. 줄의 중심은 ‘+’ , 나머지는 ‘–’로 표시한다. (중심은 40 위치로 한다.)
2. 줄다리기 줄을 초기화하는 함수 생성.
3. 줄을 당기는 함수(moveleft, moveright) 사용하며, 배열의 주소를 인자로 전달한다.
4. 현재 줄다리기 줄의 상태를 출력하는 함수 생성.
5. 'a' 또는 ';' 버튼 누르면 output 함수를 통해 줄다리기 줄의 상태 재출력.

#### __<함수 정리>__
1. input: 줄다리기 줄을 초기화하는 함수. 줄의 중앙은 '+', 나머지는 '-'로 표시.
2. moveleft: 배열의 모든 요소를 왼쪽으로 한 칸씩 이동시키는 함수.
   배열 첫번째(0번째)는 마지막(80번째) 위치로 이동한다.
3. moveright: 배열의 모든 요소를 오른쪽으로 한 칸씩 이동시키는 함수.
   배열 마지막(80번째)는 첫번째(0번째) 위치로 이동한다.
4. output: 줄다리기 줄 상태 출력 함수.
